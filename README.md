🎬 YouTube Downloader Pro

A modern desktop YouTube downloader built with Python and Tkinter.
Supports video, audio (MP3), playlists, quality selection, thumbnail preview, progress tracking, and download history.

🚀 Features

📂 Folder Selection Popup

🎵 MP3 Audio Download

🎥 Video Quality Selection (1080p / 720p / 480p / Best)

📦 Playlist Download Support

📊 Real-time Progress Bar

🖼 Thumbnail Preview

🧾 Download History Tracking

🧊 Modern Dark UI

🪟 Converted to Windows .exe

🛠 Technologies Used

Python 3

Tkinter

yt-dlp

Pillow

Requests

PyInstaller

📦 Installation (For Developers)
1️⃣ Clone Repository
git clone https://github.com/yourusername/youtube-downloader-pro.git
cd youtube-downloader-pro
2️⃣ Install Dependencies
pip install yt-dlp pillow requests
3️⃣ Run Application
python yt_pro_app.py
🪟 Convert to .EXE (Windows)
pip install pyinstaller
pyinstaller --onefile --windowed yt_pro_app.py

Executable file will be inside:

dist/yt_pro_app.exe
⚠️ Important (MP3 Support)

To enable MP3 download, you must install FFmpeg.

Download FFmpeg from:
https://ffmpeg.org/download.html

Add it to your system PATH.
